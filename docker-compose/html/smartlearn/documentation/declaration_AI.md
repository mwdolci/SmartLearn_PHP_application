Dans le cadre de ce projet, nous avons limité autant que possible l'utilisation de l'IA.
Nous l'avons utilisé de manière intelligente dans les cas suivants:
- Approfondissement de l'apprentissage (ex. pour apprendre à faire des tests unitaires en PHP que nous n'avions pas ou peu vu en cours)
- Reformulation et correction d'orthographe dans les documents
- Demande de conseils ponctuels lorsque nous avons constaté des différences entre la matière apprise en cours et la documentation sur internet (ex. require vs require_once)
